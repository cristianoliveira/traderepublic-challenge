### Frontend App

This folder contains the frontend code for the app.

### Requirements

- [Node](https://nodejs.org) 20

### Running

```bash
npm install
npm run dev
```

### CI/CD
    
```bash
npm run test # To run the tests
npm run tc # To run the type check
npm run lint # To lint the code
npm run e2e # To run the e2e tests
```
